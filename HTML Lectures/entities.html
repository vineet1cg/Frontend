HTML Entities and Symbols

HTML entities are special codes used to display reserved characters, symbols, or emojis in HTML documents. They start with an ampersand (&) and end with a semicolon (;).

What are HTML Entities?

  • Display reserved characters like <, >, and &.
  • Include currency symbols, mathematical symbols, arrows, and emojis.
  • Can be named (©), decimal (©), or hexadecimal (©).
